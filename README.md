# WGinDO

Spawns a 1Gb droplet with Debian distr in DigitalOcean and installs WireGuard VPN there.

## Installation

`pip3 install -r requirements.txt`

![do](https://raw.githubusercontent.com/Novitoll/wgindo/master/do.jpg)
